# 🐍 Snake Game (Full Stack)

A modern Snake Game built using **Frontend + Backend integration** with a dynamic UI and leaderboard system.

---

## 🚀 Features

- 🎮 Classic Snake gameplay
- ⚡ Smooth controls and movement
- 🐢 Adjustable speed (starts slow, increases gradually)
- 🌈 Advanced animated UI (CSS effects)
- 🏆 Leaderboard system (Backend connected)
- 🔄 Real-time score updates

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS (Advanced UI + Animations)
- JavaScript (Game Logic)

### Backend
- Node.js
- Express.js
